Q: Why do you think some version control system, like git, allow saving multiple files in one commit, while other, like Google docs, treat each file separately?

R: Pois o sistema git é mais "avançado", ou melhor dizendo, os programas criados/desenvolvidos são mais avançados e requerem o uso de varios files ao mesmo tempo (eg. html + css + js).

   GIT DIFF = é usado para comparar as mudanças entre dois commits (IDs).
